# PCH-2xxx-JIG-PCB
PCB for JIG on PCH-2xxx PSVITA


Nano 0805 footprint 

3.3V Version (on the 3.3V Version VCC on the vita side is left floating so you cannot charge the Vita while JIG is in use)
